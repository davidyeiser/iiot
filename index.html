Public IIoT Product Catalog

A list of products containing communication and information protocols that help automate integration.

Some rules for submissions:
  1. No products are allowed that are OPC-UA Data Access without other “advanced” OPC UA features. Despite Data Access having wide adoption and providing a level of data readability, this benefit does not provide sufficient automation of integration.
  2. No products are allowed that are plain MQTT without using a well-known communication encoding and information scheme. While MQTT helps automation integration by transporting data in a scalable way, this benefit is offset by the manual integration required to parse out the data model.
